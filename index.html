Новый курс: React Native и Expo Router

01 марта 2024 г. 925 просмотров
фото команды Автор Антон ^ link

Вышел мой новый курс!1 В этом курсе мы изучим React Native и Expo Router написав полноценное мобильное приложение с авторизацией и нативными возможностями. Курс будет максимально практическим, ведь всю теорию мы разбираем прямо во время создания проекта применяя полученные знания на практике. Мы не используем банальные ToDo приложения, а работаем с реальным API и настоящим приложении личного кабинета ученика. Этот курс подойдёт тем, кто хочет погрузиться в мобильную разработку имея опыт frontend разработки на React.

Курс максимально использует все возможности новой платформы обучения:

  • Конспекты к каждой лекции
  • Материалы и полезные ссылки
  • Скачиваемые презентации по курсу
  • Огромный проект с наставником, где будет реализовано приложения по заказу вкусного и горячего кофе ☕️

После прохождения курса вы сможете:

  1. Создавать приложения на React Native
  2. Стилизовать элементы и работать с анимацией
  3. Использовать Expo Router
  4. Создавать компоненты и layout экранов
  5. Отлаживать приложения
